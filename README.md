# Auto WoningNet

A bot that automates the process of applying for woningen on WoningNet.nl Stadsregio Amsterdam.

## Requirements
 - python (3.4+)
 - pip3
 - geckodriver 


## Installation

Install required python packages.

```bash
  pip3 install -r requirements.txt
```

## Usage

Run using python.

```bash
  python3 auto_woningnet.py
```
